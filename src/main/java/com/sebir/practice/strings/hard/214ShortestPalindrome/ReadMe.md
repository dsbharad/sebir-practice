# 214. Shortest Palindrome

URL: https://leetcode.com/problems/shortest-palindrome/description/?envType=problem-list-v2&envId=string

You are given a string s. You can convert s to a palindrome by adding characters in front of it.

_<u>Problem statement</u>_: Return the shortest palindrome you can find by performing this transformation.

 
**Example 1:**

    Input: s = "aacecaaa"
    Output: "aaacecaaa"

**Example 2:**

    Input: s = "abcd"
    Output: "dcbabcd"
 
**Constraints:**

	0 <= s.length <= 5 * 104
	s consists of lowercase English letters only.
