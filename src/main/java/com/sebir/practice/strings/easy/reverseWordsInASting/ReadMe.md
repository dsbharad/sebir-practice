# 557. Reverse Words in a String III

_<u>Problem statement</u>_: Given a string s, reverse the order of characters in each word within a sentence while still preserving whitespace and initial word order. 

**Example 1:**

    Input: s = "Let's take LeetCode contest"
    Output: "s'teL ekat edoCteeL tsetnoc"

**Example 2:**

    Input: s = "Mr Ding"
    Output: "rM gniD"
 

**Constraints:**

    1. 1 <= s.length <= 5 * 104
    2. s contains printable ASCII characters.
    3. s does not contain any leading or trailing spaces.
    4. There is at least one word in s.
    5. All the words in s are separated by a single space.
